## Overview

The AWS Cost Optimization Scanner is an enterprise-scale tool that provides:

- **30 AWS Services**: Comprehensive cost optimization coverage
- **220+ Cost Checks**: Automated analysis across all major AWS services  
- **CloudWatch Integration**: Metric-backed recommendations to prevent false positives
- **Regional Pricing**: Accurate cost calculations across all AWS regions
- **Service Filtering**: Targeted analysis for faster, focused scans
