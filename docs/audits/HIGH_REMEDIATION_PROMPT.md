### SR-1 — `reporter_phase_a.py`: Phase-A generic renderer shows the static `EstimatedSavings` string while the headline counts a different number

**Defect (verified).** The Phase-A card renderer prints each rec's free-text `EstimatedSavings` string (a percentage like `"10-30%"`/`"20-40%"`, or a hardcoded `"$316/month per endpoint"`) while the tab headline and executive summary sum the adapter's computed `EstimatedMonthlySavings` float. A FinOps reader cannot reconcile the card against the headline.

**Fix.** Single-source the per-card dollar: render the computed `EstimatedMonthlySavings` float (formatted `$/mo`, region-scaled) on the card and either drop the percentage string or relabel it as a qualitative note (not "Estimated Savings"); for `Counted=False` recs render `"$0.00/month — advisory"`. Ensure card sum == tab headline == executive-summary contribution.

**Acceptance.** For API Gateway, Glue, and Step Functions tabs, every card's displayed dollar equals its counted contribution; no card shows a percentage or a hardcoded dollar in the "Estimated Savings" slot. Reporter snapshot refreshed intentionally.

### SR-2 — `reporter_phase_b.py`: Phase-B renderers print human shim strings and sum advisory (`Counted=False`) dollars into card totals
**Repairs:** Commitment Analysis **H1**, EKS **H3**, MSK **H2**, OpenSearch **H2**, Redshift **H3** (5 services, 1 renderer family).
**Files/lines:** `reporter_phase_b.py:1102-1113` (OpenSearch), `1812-1876` + dispatch `1381` (commitment_analysis; advisory `$` at `commitment_analysis.py:282,555,638`), `1874-1875` (MSK + `services/msk.py:50,67`; Redshift + `1380-1381, 2548-2550`), `2034-2056` (`_render_eks_source`).

**Fix.** In each affected renderer path: render `rec['EstimatedMonthlySavings']` for `Counted` recs; tag `Counted=False` recs as "advisory — not added to the tab total" and suppress/zero their per-line `$`; compute every card/category total from `Counted != False` recs only, so the card total reconciles to `total_monthly_savings`. Reporter snapshot refreshed intentionally.
