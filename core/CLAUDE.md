## PricingEngine (`pricing_engine.py`)

Centralized AWS Pricing API access. Key design:

- **API endpoint**: `pricing:GetProducts` via `us-east-1` (prices are global)
- **Cache**: `PricingCache` nested class with TTL eviction (default 6 hours)
- **Fallback**: All methods return `0.0` on API failure — scans never crash due to pricing
- **Thread-safe**: Cache uses dict with key-based deduplication
